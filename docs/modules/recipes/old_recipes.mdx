## Config options

| Option key                         | Description                                                     | Type      | Default value | Possible values |
|------------------------------------|-----------------------------------------------------------------|-----------|---------------|-----------------|
| `old_recipes.enabled`     | Controls if the recipes will be registered                      | `Boolean` | `false`       | `true/false`    |
| `old_recipes.enchantedGoldenAppleRecipe` | Controls if the enchanted golden apple crafting recipe will be registered | `Boolean` | `false`        | `true/false`    |
| `old_recipes.oldGoldenAppleRecipe` | Controls if the old golden apple crafting recipe will be registered | `Boolean` | `false`        | `true/false`    |
| `old_recipes.horseArmorRecipes` | Controls if the horse armor crafting recipes will be registered | `Boolean` | `false`        | `true/false`    |
